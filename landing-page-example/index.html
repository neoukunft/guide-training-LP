⚡ Como executar seu projeto localmente?

Para não precisar atualizar a página toda vez que alterar um código, utilize o Live Server. Ele atualiza automaticamente seu navegador em tempo real!

🔌 Opção 1: Extensão do VS Code

Se você usa o Visual Studio Code, a forma mais fácil é instalar a extensão Live Server criada por Ritwick Dey. Depois é só clicar em "Go Live" na barra inferior.

Baixar Extensão VS Code →

📦 Opção 2: Pacote NPM

Se preferir usar o terminal e já tem o Node.js instalado, você pode usar o pacote npm oficial de forma global ou direta (sem baixar) via npx:

npx live-server
