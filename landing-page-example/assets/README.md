# Pasta Assets

A pasta `assets` serve para armazenar todos os recursos estáticos que compõem visualmente a sua landing page. 
O ideal é manter os recursos bem organizados em subpastas para facilitar a sua manutenção.

## Estrutura recomendada

- `images/`: para fotos, ilustrações, logos (em formatos como png, jpg, webp, svg).
- `icons/`: para os ícones utilizados (geralmente svgs ou bibliotecas de ícones).
- `fonts/`: para as fontes baixadas localmente (woff, woff2, ttf).
